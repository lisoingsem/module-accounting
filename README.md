# Accounting Module

**Accounting** - Double-Entry Accounting System

## Overview

Accounting is a Laravel module that provides a complete double-entry accounting system with general ledger, journal entries, chart of accounts, and financial reporting. Integrates with Payroll for automatic journal entry creation. Perfect for businesses requiring proper accounting, financial reporting, and compliance.
